bop
======
bop is a experimental editor that written in Rust.

Build
=====
**Notice:** Notion is compiled on Rust1.5+, so make sure rustc is up-to-date.
To build it, just run
```
cargo build
```

Installation
============
Compile it with this command and you are ready to go!
```
cargo build --release
```
